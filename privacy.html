Privacy Policy

Effective date: April 16, 2026  ·  Publisher: JoeldCo, LLC

What this app does

Open Your Windows checks local weather, air quality, and pollen conditions and returns a single recommendation: Open, Maybe, or Keep Closed. All logic runs on a server we operate; no logic runs on your device beyond displaying the result.

Location

The app requests When In Use location permission. Your coordinates are sent to our server (hosted on Supabase) to fetch conditions for your area. Coordinates are:

  • Rounded to a 0.01° grid (~1 km) before any API calls are made.
  • Never stored in a database.
  • Never associated with your identity, device ID, or any other personal information.
  • Never shared with advertisers or sold.

Analytics

We use TelemetryDeck for anonymous usage analytics. TelemetryDeck hashes all signals before they leave your device so we never see a user ID or device identifier — only aggregate counts. We collect:

  • App opens
  • Decision type displayed (Open / Maybe / Closed) and confidence level
  • Refresh source (pull-to-refresh vs. automatic)
  • Pollen sensitivity toggle state (on/off)
  • Error type when an error is displayed

No location data is ever included in analytics signals.

Third-party data sources

To generate a recommendation we query these services with your rounded coordinates:

  • OpenWeatherMap — weather and forecast
  • AirNow (EPA) — air quality index
  • Google Pollen API — pollen forecast

Each provider's own privacy policy governs their handling of query parameters.

No accounts, no persistent storage

Open Your Windows has no user accounts. We do not store any personal data on our servers. Your pollen sensitivity preference is stored only on your device using iOS standard storage.

Children

This app is not directed at children under 13 and does not knowingly collect data from them.

Changes

If this policy changes materially, we will update the effective date above. Continued use of the app after a change constitutes acceptance.
